프로젝트 기간 : 23.01.08 ~ 23.~
프로젝트 방식 : Spring MVC

개발 인원 : 2명

개발 도구 : Spring-tool-suite(sts-3.9.11.RELEASE),

개발 환경 :

javascript, jquery, MyBatis 3.3.0, JSTL 1.2, Maven 4.0.0, aspectjweaver 1.8.8
형상관리 툴 : GitHub
프로젝트 설명

인터넷 강의를 들으며 Spring framework 를 공부했고
포케리움은 Spring MVC기반으로 제작중인 웹사이트입니다.
웹사이트 기능으로는
회원관련 - 가입/탈퇴, 로그인/로그아웃, 정보수정
상품관련 - 상품목록/상품상세정보
주문관련 - 주문/장바구니/결제
마이페이지 - 마이페이지/회원정보수정/주문조회
커뮤니티 - 자유게시판/자랑게시판 글쓰기/수정/삭제
댓글 및 대댓글쓰기/수정/삭제
포켓몬도감 - 포켓몬목록/상세정보조회
입니다.
제가 맡아서 개발을 진행 중인 부분은
DB모델링, 회원관련, 자유게시판 관련된 기능 입니다.
회원 기능중 Aop를 통해 비밀번호 를 SHA-256 으로 암호화 하였고, 회원가입/수정/로그인 기능에 적용시켰습니다.
자유게시판은 웹의 기본이되는 CRUD 방식으로 개발을 진행하였고, 현재 댓글 및 대댓글 기능을 개발하고 있습니다.
